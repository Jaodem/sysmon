# Sysmon

Sistema de monitoreo de procesos locales en tiempo real

## Stack

- **Backend:** Python + FastAPI + psutil (tipado con Pydantic)
- **Database:** PostgresSQL
- **Frontend:** Next.js + TypeScript + Tailwind CSS + shadcn
- **Infraestructura:** Docker + Docker Compose
